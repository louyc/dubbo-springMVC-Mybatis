注册协议提示

您在申请注册流程中点击同意前，应当认真阅读《万物语联普通用户服务协议》和《法律声明及隐私权政策》。请您务必阅读并充分理解相关条款内容，其中包括：

1、与您约定免除或限制责任的条款；

2、与您约定法律适用和管辖的条款；

3、其他以粗体下划线标识的重要条款。

如您未满18周岁，或以其他形式被限制民事行为能力，请在监护人的陪同下阅读协议。如您对协议有任何疑问，可向万物语联客服咨询。

【特别提示】当您勾选“我同意接受《万物语联普通用户服务协议》和《法律声明及隐私权政策》全部条款”并点击“确认”按钮后，即表示您已接受前述协议的全部内容并同意与万物语联订约。如您因万物语联提供的服务与万物语联发生争议的，适用《万物语联普通用户服务协议》处理。如您在使用万物语联提供的服务的过程中与其他用户发生争议的，依您与其他用户达成的协议处理。

阅读协议的过程中，如果您不同意相关协议或其中任何条款约定，您应立即停止注册程序。
